# ADR-001

Title:
Gemini 2.5 Flash for Vision Analysis

Date:
2026-06-25

Status:
Accepted

Problem:
Need fast, accurate image classification for civic infrastructure photos.

Decision:
Use Gemini 2.5 Flash via Supabase Edge Function.

Reason:
Fast multimodal inference, structured JSON output, cost-effective for hackathon.

Alternatives:
- GPT-4o (slower, more expensive)
- Claude Vision (good quality, less hackathon familiarity)

Consequences:
- Google API dependency
- Edge function needed to protect API key
- Fast response times for demo flow
